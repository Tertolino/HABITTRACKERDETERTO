# 🚀 Habit Tracker Pro

Um rastreador de hábitos e tarefas moderno, desenvolvido com foco em organização pessoal e produtividade. Este projeto permite gerir tarefas com níveis de urgência, categorias personalizadas e notificações de atraso, tudo com suporte a modo escuro e claro.

## 📋 Funcionalidades

- **Gestão de Tarefas**: Adicione, edite e remova tarefas facilmente.
- **Categorização**: Organize as suas atividades em:
  - Social, Financeiro, Estudo, Trabalho, Saúde, etc.
- **Nível de Urgência**: Controle visual de prioridade (0% a 100%).
- **Controle de Tempo**: Agendamento de data e hora com alertas de tarefas atrasadas.
- **Modo Dark/Light**: Interface adaptável à preferência do utilizador.
- **Paginação**: Navegação fluida entre listas de tarefas extensas.
- **Notificações**: Alertas no navegador para manter o foco no prazo.
- **Persistência de Dados**: As tarefas ficam guardadas no `localStorage` do navegador.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização moderna com variáveis (CSS Variables) para os temas.
- **JavaScript (Vanilla)**: Lógica de manipulação do DOM e gestão de datas.
- **Vercel**: Hospedagem e deploy contínuo.
- **GitHub**: Controlo de versão e repositório.
